# Conceito Sistema Bancário 🏦 - Java POO

Este projeto demonstra conceitos básicos de **Programação Orientada a Objetos (POO)** em **Java**, através da criação de uma classe que representa uma conta bancária.

---

## 🔍 Sobre o projeto
O projeto consiste na criação de uma classe `Conta` com atributos como saldo, dono, número da conta, tipo e etc. Também inclui métodos para sacar, depositar, abrir conta e etc.
Uma classe de teste (`Contas`) é utilizada para instanciar objetos e testar as funcionalidades da classe `Conta`.

---

## 📊 Tecnologias utilizadas
- Java
- IntelliJ

---

- Uso de métodos e atributos
